# Plots for WGCNA

Visualization for results from
[`glystats::gly_wgcna()`](https://glycoverse.github.io/glystats/reference/gly_wgcna.html)
(`glystats_wgcna_res` objects). Draw a heatmap of eigenvalues.

## Usage

``` r
# S3 method for class 'glystats_wgcna_res'
autoplot(object, ...)
```

## Arguments

- object:

  A `glystats_wgcna_res` object.

- ...:

  Ignored.

## Value

A ggplot object.
